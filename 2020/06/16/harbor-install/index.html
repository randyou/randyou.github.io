安装准备

把解压出来的 harbor 文件夹放到你想安装的地方，比如 /usr/local/harbor

需要修改以下内容

  • hostname：修改成需要使用的域名，为了能外部访问，不要使用 127.0.0.1 或 localhost

  • https 配置如果没有可以先注释掉

  • data_volume：文件存储地址
