Dockerfile 备忘

2020-06-13 Dockerfile, docker 0 评论

使用多阶段构建

多阶段构建有效使用镜像层级缓存，加快构建速度

  • 优先添加变化频率更低的文件，提升缓存命中率
  • 最终镜像中不包含构建阶段使用的工具，大量减小镜像体积

使用更小的基础镜像

更小的基础镜像有效节省大量磁盘空间，提高传输效率

  • 使用 alpine 文件系统通常能获得更小的镜像
